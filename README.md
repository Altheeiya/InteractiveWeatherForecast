# InteractiveWeatherForecast

InteractiveWeatherForecast adalah aplikasi dashboard cuaca berbasis web yang dirancang dengan antarmuka bergaya retro pixel art. Aplikasi ini menyediakan informasi cuaca real-time dan prediksi cuaca untuk 5 hari ke depan menggunakan data dari OpenWeatherMap API.

Fitur utama meliputi pencarian lokasi berdasarkan nama kota, kemampuan menyimpan daftar kota favorit, serta opsi interaktif untuk mengubah tema tampilan (Light/Dark Mode) dan satuan suhu (Celsius/Fahrenheit). Proyek ini dibangun menggunakan HTML, CSS, dan JavaScript.
